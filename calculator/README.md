# 簡易計算機

## 功能

1.使用簡單的四則運算

## 技術手段

1.運用result.innerText = eval(a.value + cal.value + b.value)計算

2.利用document.getElementById('')取得元件

3.利用<link rel="stylesheet" type="text/css" href="cal.css">將CSS套用至HTML

## 說明

1.輸入'a','b'兩數值及'cal'運算符號

2.計算"a+cal+b"

3.輸出result

## 舉例

輸入 a = 6 , b = 12 , cal = "+"

計算 6 + 12

按下按鈕後計算得result = 18
